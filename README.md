# wrASD_bot
Автоматический бот Telegram для удаление фишинга с использованием машинного обучения

# Инструкция по запуску
## 0. Скопируйте репозитрий
```git clone https://github.com/SemkaShr/wrASD_bot```
## 1. Установка зависимостей
```pip install -r requirements.txt```
## 2. Обучение модели
Запустите файл train_model.py:
```python train_model.py```
## 3. Запуск бота
1. Обновите BOT_TOKEN в файле config.py
2. Запустите файл main.py
```python main.py```

Готово!
